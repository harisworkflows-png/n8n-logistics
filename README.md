# n8n Logistics Automation

Automation workflows built for freight forwarders and logistics teams.

---

## Current Workflow

### Invoice Discrepancy Alert

Detects mismatches between quoted totals and invoiced totals.

When a variance is detected:
- Calculates difference
- Flags discrepancy
- Triggers alert notification

Location:
`workflows/invoice-discrepancy-alert.json`

---

## How To Use

1. Download the JSON file
2. Import into n8n
3. Configure credentials
4. Activate workflow
